## 可生成视图

当前只定义以上输出名称和来源边界，本轮不实现生成器，也不创建看似真实的空结果。未来生成内容必须可从正式对象重建，并标明生成时间、输入版本/hash 和未覆盖范围。

## 声明边界

本目录不是需求、功能树、Spec、Task 或影响关系权威。删除派生视图不得删除事实；编辑视图不得修改 approved Requirement Baseline。`必需`、`条件启用` 和 `运行时生成` 只表示目录生命周期，不表示自动化已经实现。
