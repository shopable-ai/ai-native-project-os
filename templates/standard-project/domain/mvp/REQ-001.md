# REQ-001：交付对象必须可追溯

项目必须为每个交付对象记录稳定编号、批准需求、验收判据和 Evidence 出口，使审查者能够从交付对象回溯到批准事实。

## 验收方向

- Spec 五件套存在且职责分离；
- `traceability.md` 指向 `domain/` 中的批准对象；
- review evidence 明确证明范围和禁止外推项。
